# Wireframes

Reference the **Creating an Entity Relationship Diagram** final project guide in the course portal for more information about completing this deliverable.

## List of Pages

* ⭐ Landing Page
* ⭐ User Dashboard Page
* ⭐ Credit Card Details Page
* Sign Up / Login Page
* All Favorites Page
* All Reviews Page
* Browse Cards Page

## Wireframe 1: Landing Page

The landing page introduces CardMaxer and allows users to begin browsing credit cards or sign in to their account.

![CardMaxer landing page wireframe](https://raw.githubusercontent.com/9Mohamedismail/web103_finalproject/main/planning/landing.png)

## Wireframe 2: User Profile Dashboard

The User Profile Dashboard gives users an overview of their account in one organized page. It displays their credit score, favorited credit cards, and previously submitted reviews. Users can select a favorited card or past review to view more details.

![CardMaxer dashboard wireframe](https://raw.githubusercontent.com/9Mohamedismail/web103_finalproject/main/planning/dashboard.png)

## Wireframe 3: Credit Card Details Page

The details page displays information about a selected credit card, including its rewards, fees, benefits, drawbacks, recommended credit score, and user reviews.

![CardMaxer credit card details wireframe](https://raw.githubusercontent.com/9Mohamedismail/web103_finalproject/main/planning/details.png)
