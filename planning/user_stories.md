# User Stories

## Outline User Roles

**Credit Card Shopper:** A user who wants to browse, compare, review, and find credit cards that match their credit score and spending interests.

## Draft User Stories

1. As a credit card shopper, I want to browse a list of credit cards so that I can see what options are available.

2. As a credit card shopper, I want to view a credit card’s details so that I can understand its rewards, fees, benefits, and drawbacks.

3. As a credit card shopper, I want to filter cards by category so that I can find cards for travel, cash back, groceries, gas, or building credit.

4. As a credit card shopper, I want to sort cards by annual fee or recommended credit score so that I can compare cards more easily.

5. As a credit card shopper, I want to enter my credit score range so that I can get card recommendations that fit me.

6. As a credit card shopper, I want to favorite credit cards so that I can save the options I am most interested in.

7. As a credit card shopper, I want to remove a card from my comparison list so that I can keep only the cards I am seriously considering.

8. As a credit card shopper, I want to create a review for a credit card so that I can share my experience with other users.

9. As a credit card shopper, I want to edit my review so that I can update my opinion if my experience changes.

10. As a credit card shopper, I want to delete my review so that I can remove information I no longer want to share.

11. As a credit card shopper, I want to see reviews from other users so that I can learn about real experiences with each card.
